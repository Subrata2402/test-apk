TestAPK

Securely upload, manage and distribute Android APKs using your own Google Drive.

TestAPK is designed for Android developers and QA teams who need a secure way to privately distribute APK files to authorized testers without relying on third-party hosting services.

What is TestAPK?

TestAPK is a secure Android application distribution platform built for software developers and QA teams.

For Developers

Upload Android APK files through the TestAPK web dashboard. Every APK is stored directly inside your own Google Drive, giving you complete ownership and control of your releases.

For Testers

Testers can securely browse available builds, read release notes, download APKs, and install applications directly from the TestAPK Android app.

How TestAPK Works

1

Google Sign-In

Developer securely signs in using Google OAuth.

2

Connect Google Drive

TestAPK creates a dedicated application folder.

3

Upload APK

APKs are uploaded securely to the developer's own Drive.

4

Invite Testers

Authorized testers receive access to the releases.

5

Install App

Testers download and install builds directly from the companion Android application.

Why Google Sign-In is Required

TestAPK uses Google Sign-In only for authentication and account management.

Information We Access

  • Name
  • Email Address
  • Profile Picture

Why We Need It

  • Authenticate developers
  • Identify user accounts
  • Personalize dashboard
  • Connect Google Drive

Why Google Drive Access is Required

TestAPK requests only the Google Drive drive.file permission.

What TestAPK Can Do

  • Create a dedicated testapk-releases folder
  • Upload APK files
  • Read APKs uploaded by TestAPK
  • Delete APKs uploaded through TestAPK

What TestAPK Cannot Do

  • Read personal Google Drive files
  • Access unrelated folders
  • Modify existing documents
  • Share files without your permission

Security & Privacy

🔒 Google OAuth

Secure authentication using Google's OAuth 2.0 platform.

🛡 HTTPS Encryption

All communication between the application and server is encrypted using HTTPS.

✔ SHA-256 Verification

APK integrity is verified using SHA-256 checksums before installation.

☁ Your Google Drive

APK files remain inside your own Google Drive account. TestAPK does not host your APKs.
